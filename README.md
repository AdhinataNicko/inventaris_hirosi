# Sistem Inventaris Barang HIROSI

## Deskripsi Project
Sistem Inventaris Barang HIROSI merupakan aplikasi web yang dikembangkan untuk membantu pengelolaan data inventaris barang pada organisasi HIROSI. Sistem ini dirancang untuk memudahkan proses pencatatan, pengelolaan, dan pemantauan kondisi barang secara terstruktur dan efisien.

Aplikasi ini dikembangkan sebagai Project Akhir mata kuliah Pemrograman Web.

## Tujuan Pengembangan
Tujuan dari pengembangan Sistem Inventaris Barang HIROSI adalah:
- Mempermudah pengelolaan data inventaris barang
- Mengurangi kesalahan pencatatan data barang
- Mengimplementasikan konsep CRUD dalam pemrograman web
- Memenuhi tugas Project Akhir mata kuliah Pemrograman Web

---

## Ruang Lingkup Sistem
Ruang lingkup sistem ini meliputi:
- Pengelolaan data inventaris barang
- Pengolahan data barang menggunakan fitur CRUD
- Penyimpanan dan penampilan data barang dalam sistem berbasis web

---

## Fitur Sistem
Sistem Inventaris Barang HIROSI memiliki fitur utama sebagai berikut:
- Menambahkan data barang
- Menampilkan data barang
- Mengubah data barang
- Menghapus data barang

Data barang yang dikelola meliputi:
- Nama Barang
- Jumlah Barang
- Kondisi Barang
- Lokasi Barang
